# bada 프로젝트

## 개요

멘토와 멘티가 함께 알고리즘 문제를 푸는 것에 대해 생각으로 시작하게 되었다.

[참고: 구름.io](https://level.goorm.io)

## 목표

이 프로젝트를 통해서 경험하지 못한것에 도전해 본다.

- 코드 편집기 구현
- 영상 처리 구현
- 다중 접속 및 다중 편집
- 네트워크에 집중하고자 한다.
- 혼자 개발이 아닌 여러사람과 협업이라는 관점을 가지고 프로젝트에 임한다.

## 기술

- 파이썬 3.11
- fastapi
- mysql
- docker
- github action
- naver cloud
